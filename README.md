# SwiftHTTPManager
Swift网络请求模块封装

MANetWork is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MANetWork', :git => 'https://github.com/fengyunjue/SwiftHTTPManager.git'
```
